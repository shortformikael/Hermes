# Hermes
Lightweight Go Binary utilized as message redirect to your telegram account. Meant for Linux Servers. 

Current Personal Use:

Notification System for Servers to send alerts on error and daily server Health.

# Install:

A user should be able to just call the bash script with cURL and have the installation script handle everything. Once installed to /usr/local/bin/, open the TUI and have a "Start-up" procedure setup the necessary information.

# Instructions

Once finished: You can start the program by calling ´hermes´ in the console.
